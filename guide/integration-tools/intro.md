# Integrar herramientas

Las herramientas de Biteral, como el [buscador natural](https://biteral.net/natural-search), las [recomendaciones personalizadas](https://biteral.net/personalized-recommend), el [análisis de tendencias](https://biteral.net/trend-analysis) o las [etiquetas de cliente](http://biteral.net/profile-tags) pueden integrarse manualmente en vuestro sistema utilzando la API o el SDK, lo que permite crear integraciones personalizadas según vuestras necesidades.

!!!
La API y el SDK también permiten integrar la inteligencia artificial de Biteral de forma más profunda en vuestros procesos, e incluso desarrollar nuevas herramientas basadas en IA adaptadas específicamente a vuestro modelo de negocio.
!!!

### Componentes listos para producción
También ofrecemos componentes listos para usar que permiten incorporar algunas de estas herramientas con solo copiar y pegar unas pocas líneas de código.
